1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/jkourashi/jkourashi.github.io.git
   ```

2. Navigate into the directory:
   ```bash
   cd jkourashi.github.io
   ```

3. Install dependencies (if any).
   - You may need to install certain packages or libraries depending on your project setup.

4. Run the project locally:
   ```bash
   // Command to start your server or build your project.
   // Example: npm start
   ```

5. Deploy your changes to GitHub Pages:
   - Push your updates to the `main` branch:
   ```bash
   git add .
   git commit -m "Your commit message"
   git push origin main
   ```
   - GitHub Pages should automatically update your site based on the content of the `main` branch.
